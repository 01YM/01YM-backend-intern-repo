## Reflection 

### Which AI tools did you try 
- ChatGPT
- Claude AI
- Github Copilot

### What worked well? what didnt?

- Claude AI: worked best for planning the code architecture and giving thoughtful detailed explanations. It feels the most useful when it comes to writing code and I find it more useful then chatgpt. 

- ChatGPT: Great for Ideas, explaining concepts, debugging and giving step by step fixes, guidelienes and solutions. 

- Github Copilot: Worked well to speed up the coding process and suggest functions, methods, fill out code real time while typing. It works well when you need to quickly generate basic functions and code. Not good for planning out the structure.

### When do you think AI is most used for coding ?

- Its the most useful for autofilling repetitive code snippets and planning out, giving a skeleton to work with or when you want to start coding from scratch. Its also useful to write test cases or debugging smaller issues, while going over new concepts and exploring different approaches to a problem.
