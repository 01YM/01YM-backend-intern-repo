# VS Code Productivity

## Top VS Code Shortcuts

| Shortcut | Action |
|----------|---------|
| Ctrl + P | Quickly open a file by name |
| Ctrl + Shift + P | Open Command Palette |
| Ctrl + / | Comment or uncomment a line |
| Alt + Up / Down | Move a line up or down |
| Ctrl + D | Select the next word |
| Ctrl + F | Find text in current file |
| Ctrl + H | Find and replace text |
| Ctrl + Shift + F | Search across all files |
| Ctrl + ` | Open or close terminal |
| Ctrl + B | Show or hide sidebar |
| Ctrl + Shift + N | Open a new VS Code window |
|----------|---------|
| Ctrl + N | Create a new file |
| Ctrl + S | Save current file |
| Ctrl + Shift + S | Save As |
| Ctrl + W | Close current tab |
| Ctrl + Tab | Switch between open tabs |
| Ctrl + Shift + Tab | Switch tabs backwards |
| Ctrl + K Ctrl + C | Comment selected lines |
| Ctrl + K Ctrl + U | Uncomment selected lines |
| Ctrl + L | Select current line |
| Ctrl + Space | Trigger suggestions/autocomplete |
| Shift + Alt + F | Format document |
| Ctrl + K Ctrl + F | Format selected code |

## Which shortcuts will be most useful in daily work?

Shortcuts for saving files, opening fiels by name, selecting lines, copy pasting, finding text, commenting and uncommenting lines.

## How can keyboard shortcuts improve productivity?

  moving between the keyboard and mouse takes extra time, learning shortcuts will make coding faster and helps maintain focus. They are really useful for big projects where searching, navigating, formatting, and debugging happen very often.
