# 🔐 Data Privacy & Confidentiality Reflection 

## Steps to take for handling data securely 
1. Using approved tools only
2. Do not download unnecessary data
3. Keep information private
4. Lock the device and use strong passwords
5. Double check before sharing information 
6. Being careful and consistent to prevent risks 
## Storing, sharing and disposing sensitive information 
- Sensitive data cannot be stored in personal devices. It can only be shared with other autherized team members through safe channels. Data needs to be deleted correctly once no longer in need. 
## Common mistakes 
- Sending data to wrong people
- Uploading it to AI 
- Using weak passwords 

## Habits to improve data security 
- Before sending or uploading sensitive data anywhere, take a pause and review everything. This will reduce the risks of human error. 

## Key Learning/ security measure
- Think twice before uploading data anywhere, there could be serious consequences for a small mistake. Always follow company guidelines and secure practices.
